AVAILABLE API FUNCTIONS (All in lib/api.ts)

Statistics:
  getStats()                          → { activeListingCount: number }

Browse:
  getDistricts()                      → District[]
  getListings(filters)                → PagedListings
  getListing(slug)                    → ListingDetail
  getSimilar(slug)                    → ListingCard[]

Authentication:
  login(email, password)              → { token, user }
  register(data)                      → { token, user }
  getMe()                             → User | null
  logout()                            → void

Listings Management:
  createListing(data)                 → { id, slug }
  getMyListings(status?)              → ListingCard[]

Media & AI:
  uploadMedia(files)                  → { ids: string[] }
  generateDescription(data)           → { description }

Contact:
  submitLead(listingId, data)         → { id }

KEY PATTERNS FOR DEVELOPERS

Server Component (for pages with data fetching):
  export default async function Page() {
    try {
      const data = await getListings({ type: 'sale' });
      return <Component data={data} />;
    } catch (error) {
      return <FallbackUI />;
    }
  }

Client Component (for forms and interactivity):
  'use client';
  
  useEffect(() => {
    const user = await getMe();
    if (!user) router.push('/dang-nhap');
  }, []);

Error Handling:
  try {
    await apiCall();
  } catch (err: any) {
    displayError(err.message); // Vietnamese from backend
    if (err.status === 401) router.push('/dang-nhap');
  }
